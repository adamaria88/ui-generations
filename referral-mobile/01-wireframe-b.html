Idenya

Halaman ini punya kamu, bukan brosur program. Yang paling atas = duit yang udah kamu dapet.

Angka program (Rp500rb) turun jadi target di tangga tier, bukan judul.

Cocok buat

User aktif — yang udah punya Mitra dan mau ngecek progres. Ini yang bikin halaman dibuka berulang.

Ref: Satispay (riwayat per milestone), Brex (tab tracking), N26 (Earned vs Pending).

Risikonya — lihat layar 3

User baru disambut "Rp0". Itu kebalikan dari memikat. Padahal mayoritas user yang buka pertama kali ada di kondisi ini.

Tangga tier semua kosong, riwayat kosong. Halaman yang harusnya ngajak malah kelihatan seperti laporan kosong.

Kenapa C jadi rekomendasi

C naruh angka program di atas (hook buat user baru) lalu progres di bawahnya (buat user aktif) — dan untuk user baru, blok progres diganti "Cara kerjanya".

Jadi dua-duanya kebagian tanpa ada yang disambut Rp0.
