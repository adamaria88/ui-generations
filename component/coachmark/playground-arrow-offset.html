Arrow Offset — konsep

Posisi layer nested nggak bisa di-override di instance Figma, tapi padding bisa (udah diuji). Jadi arrow ditaruh di dalam auto-layout row dan digeser lewat padding-left — bikin posisinya bebas, bukan cuma 3 titik.

Target yang disorot

Cara posisi arrow

Preset = varian Start/Center/End yang sekarang. Offset bebas = atur padding-left per instance.

Arrow offset

Rentang aman 16–292. Di bawah 16 arrow nabrak sudut membulat (radius 12); di atas 292 nyundul sisi kanan.
